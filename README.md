# Malaysia Car Popularity Analysis

## Project Overview
This project analyzes vehicle registration trends in Malaysia using open government data. The goal is to explore the most popular car brands and models and understand market dominance.

## Data Source
Vehicle registration data obtained from Malaysia’s official open data portal: data.gov.my

## Dataset
The dataset includes:
- Car models registered from 2020–2025
- Registration counts
- Yearly trends

## Planned Analysis
- Market share analysis of car brands
- Most popular car models
- Trend analysis over time
- Comparison of local vs foreign brands

## Tools Used
- Microsoft Excel (data preparation)
- Microsoft Power BI
- GitHub (project documentation)
